# Melda-n-n-Maceralar-A-Personal-Learning-Journey
This is a personal, non-commercial RPG project created as a hands-on exercise to explore game development logic in C#. It’s a tribute to the classic top-down adventure games, built specifically to practice mechanics like state machines, UI synchronization, and enemy AI.

    Note: This project was developed for educational purposes to consolidate my knowledge of C# and game design patterns. While building it, I followed various tutorials and community resources to understand best practices.
